# 快捷键

command + option +J 快速定位到当前打开的类所在位置目录结构
command + 1.....7  切换工程选项卡左边的那个
ESC 代码自动补全
command + shift +k  clean 工程
command + k 清空console 输出
command + f 搜索当前文件
command + shift + f 搜索整个工程
command + l 跳转到某一行
command + control + 左右方向键  前进或者后退已打开过的文件
command + control + 上下方向键 切换.h 和 .m文件
command + 0 隐藏/显示导航栏
