# html总结

- 简介

```html
<!DOCTYPE html>
<html>
<body>

<h1>我的第一个标题</h1>
	<h2>我的第二个标题</h2>

<p>我的第一个段落。</p>

</body>
</html>
```

- 注意：DOCTYPE 声明了文档类型

### HTML 标题
- h1 - h6 来表示标题的大小

```html
<!DOCTYPE html>
<html>
<body>

<h1>这是标题 1</h1>
<h2>这是标题 2</h2>
<h3>这是标题 3</h3>
<h4>这是标题 4</h4>
<h5>这是标题 5</h5>
<h6>这是标题 6</h6>

</body>
</html>
```

### 段落


```html
<!DOCTYPE html>
<html>
<body>

<p>这是一个段落。</p>
<p>这是一个段落。</p>
<p>这是一个段落。自动换行</p>

</body>
</html>
```
